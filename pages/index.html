Phoenix Channel Runtime

Rust APIs for Phoenix Channels v2 on native and browser targets.

  • Managed client Sockets, channels, reconnects, Presence, and telemetry
  • Native runtime Tokio WebSocket transport and Send + Sync worker handles
  • Browser runtime WebSocket, LongPoll fallback, and browser lifecycle integration
  • Protocol runtime Phoenix v2 codecs, state machine, sessions, and Presence sync
